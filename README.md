# Max Raffle Bot

Max Raffle Bot — сервис для проведения розыгрышей и рассылок через MAX, Telegram и VK.

Проект объединяет админ-панель, чат-ботов и мини-приложение для участников. Через админку можно создавать розыгрыши, загружать промокоды, отправлять сообщения, работать с пользователями и синхронизировать данные с внешними каналами.

## Возможности

- создание и управление розыгрышами;
- работа с участниками через MAX, Telegram и VK;
- webhook-обработчики для разных мессенджеров;
- админ-панель для управления контентом и настройками;
- импорт и выдача промокодов;
- рассылки по пользователям;
- мини-приложение для участия в розыгрыше;
- интеграция с Google Sheets;
- ведение истории сообщений и действий пользователей.

## Из чего состоит сервис

- `admin/` — административная панель;
- `raffle_app/` — пользовательское мини-приложение розыгрыша;
- `lib/` — работа с API, базой данных, рассылками, чатами и промокодами;
- `webhook.php` — webhook для MAX;
- `telegram_webhook.php` — webhook для Telegram;
- `vk_webhook.php` — webhook для VK;
- `cron.php` и `broadcast_cron.php` — фоновые задачи;
- `tools/` — служебные скрипты;
- `google_sheets/` — интеграция с Google Sheets.

## Технологии

PHP, MySQL, MAX Bot API, Telegram Bot API, VK API, Google Sheets API, JavaScript, HTML/CSS, cron.
